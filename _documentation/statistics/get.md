## {{page.title}}

```
GET /statistics/:userid
```

### Parameters


| Key  |       |  Description |
|:------------|:-------------:|:------|
| `username`   | *optional* | Username. |
| `first_name`       | *required* | First name. |
| `url` | *optional* | Portfolio/personal URL.|

### Response

```javascript
200 OK
X-Ratelimit-Limit: 1000
X-Ratelimit-Remaining: 999
```
